# Create a New Card

**URL** : `/cards`

**Method** : `POST`

## Success Response

**Code** : `201 CREATED`

**Content example**

```json
{
    "number": "9001",
    "name": "Black Lotus",
    "colors": [],
    "manaCost": "{0}",
    "manaValue": 0,
    "types": [],
    "subtypes": [],
    "keywords": [],
    "text": "{T}, Sacrifice Black Lotus: Add three mana of any one color.",
    "artist": "Christopher Rush",
    "_id": "",
    "__v": 0
}
```
